# 图片资源说明

## TabBar 图标
需要在 `images/tabbar/` 目录下放置以下图标文件：
- `kitchen.png` - 厨房图标（未选中）
- `kitchen-active.png` - 厨房图标（选中）
- `order.png` - 订单图标（未选中）
- `order-active.png` - 订单图标（选中）
- `profile.png` - 个人中心图标（未选中）
- `profile-active.png` - 个人中心图标（选中）

## 默认图片
- `default-avatar.png` - 默认头像
- `default-bg.jpg` - 默认背景图
- `default-recipe.png` - 默认菜谱图片

注意：这些图片需要您自行添加，或者使用在线图片链接。
