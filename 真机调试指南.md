## 真机调试步骤

### 方法 1：开启真机调试（推荐）

1. **在手机上打开小程序**
2. **点击右上角"..."按钮**
3. **选择"打开调试"**
4. **重新进入小程序**
5. **查看控制台错误信息**

### 方法 2：查看开发者工具控制台

1. **在微信开发者工具中**
2. **点击"预览"按钮**
3. **扫码在手机上打开**
4. **回到开发者工具，查看"调试器"面板**
5. **查看 Console 标签页的错误信息**

### 方法 3：检查网络请求

1. **在开发者工具的"调试器"面板**
2. **切换到"Network"标签**
3. **查看是否有请求失败**

## 常见白屏原因

### 1. JavaScript 错误
- **症状**：页面完全空白
- **解决**：查看控制台错误，修复代码

### 2. 数据未初始化
- **症状**：页面有部分内容但不完整
- **解决**：检查 `app.js` 的 `onLaunch` 是否正常执行

### 3. 样式问题
- **症状**：内容存在但不可见
- **解决**：检查 CSS 样式，确保有背景色

### 4. 基础库版本不兼容
- **症状**：某些功能不工作
- **解决**：在开发者工具中设置基础库版本为稳定版本

## 如果仍然白屏

1. **打开真机调试**，查看具体错误信息
2. **检查控制台**，找到第一个报错
3. **根据错误信息**，定位问题代码
4. **修复后重新编译**测试
